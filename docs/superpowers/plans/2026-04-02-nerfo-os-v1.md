# NerfOS v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build two bootable, installable ISOs (`nerf-os-1.0-x86_64.iso` and `nerf-os-1.0-i686.iso`) that boot in under 20 seconds on HDD, idle at ~120–160MB RAM in GUI mode, and run on 2008–2015 hardware.

**Architecture:** Debian 13 Trixie minimal bootstrapped with `debootstrap`, runit init pulled from antiX 26 repositories, built into ISOs with `live-build`. Two boot modes (GUI: Openbox + tint2, Terminal: bash) selected via GRUB kernel parameter `nerf.mode=`. A `nerf-optim` script runs at first boot to tune the system for the detected hardware. A `nerf-install` script provides a CLI-guided installation to HDD.

**Tech Stack:** debootstrap, live-build, runit, Openbox, tint2, PCManFM, Firefox-ESR, Midori, zram, ufw, lightdm, GRUB2, bash

---

> **Build host requirement:** live-build and debootstrap require Linux. All build tasks run inside WSL2 (Debian 12 Bookworm or Debian 13 Trixie) on the Windows host. The repo at `D:\Repos\Github\nerf-os` is accessible from WSL2 at `/mnt/d/Repos/Github/nerf-os`.

  ```bash
  #!/bin/bash
  PASS=0

  assert_contains() {
    local haystack="$1" needle="$2" msg="$3"
    if echo "$haystack" | grep -q "$needle"; then
      echo "PASS: $msg"
      PASS=$((PASS+1))
    else
      echo "FAIL: $msg"
      echo "  Expected to find: $needle"
      echo "  In: $haystack"
      FAIL=$((FAIL+1))
    fi
  }

  assert_equals() {
    local actual="$1" expected="$2" msg="$3"
    if [ "$actual" = "$expected" ]; then
      echo "PASS: $msg"
      PASS=$((PASS+1))
    else
      echo "FAIL: $msg"
      echo "  Expected: $expected"
      echo "  Actual:   $actual"
      FAIL=$((FAIL+1))
    fi
  }

  assert_file_exists() {
    local file="$1" msg="$2"
    if [ -f "$file" ]; then
      echo "PASS: $msg"
      PASS=$((PASS+1))
    else
      echo "FAIL: $msg — file not found: $file"
      FAIL=$((FAIL+1))
    fi
  }

  run_tests() {
    for test_fn in "$@"; do
      $test_fn
    done
    echo ""
    echo "Results: $PASS passed, $FAIL failed"
    [ "$FAIL" -eq 0 ]
  }
  ```

- [ ] **Step 5: Write build.sh**

  Create `build/build.sh`:
  ```bash
  #!/bin/bash
  set -euo pipefail

  ARCH="${1:-}"
  SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
  REPO_ROOT="$(dirname "$SCRIPT_DIR")"
  BUILD_DIR="$SCRIPT_DIR"
  VERSION="1.0"
  OUTPUT="$REPO_ROOT/output"

  usage() {
    echo "Usage: $0 [x86_64|i686]"
    echo ""
    echo "Builds NerfOS ISO for the specified architecture."
    echo ""
    echo "Examples:"
    echo "  $0 x86_64    # Build 64-bit ISO"
    echo "  $0 i686      # Build 32-bit ISO"
    exit 1
  }

  if [ -z "$ARCH" ]; then
    usage
  fi

  if [ "$ARCH" != "x86_64" ] && [ "$ARCH" != "i686" ]; then
    echo "Error: unsupported architecture '$ARCH'. Use x86_64 or i686."
    exit 1
  fi

  if [ "$(id -u)" -ne 0 ]; then
    echo "Error: build.sh must be run as root (required by live-build/debootstrap)."
    exit 1
  fi

  echo "==> Building NerfOS $VERSION for $ARCH"
  mkdir -p "$OUTPUT"

  WORK_DIR="$(mktemp -d /tmp/nerfo-build-XXXXXX)"
  trap 'rm -rf "$WORK_DIR"' EXIT

  cd "$WORK_DIR"

  # Copy live-build config
  cp -r "$BUILD_DIR/config" .

  # Merge arch-specific overrides
  if [ -d "$BUILD_DIR/config/$ARCH" ]; then
    cp -r "$BUILD_DIR/config/$ARCH/." config/
  fi

  # Copy bootloader configs
  cp -r "$BUILD_DIR/bootloaders/grub" config/bootloaders/ 2>/dev/null || true

  # Set architecture
  lb config \
    --architecture "$ARCH" \
    --distribution trixie \
    --archive-areas "main contrib non-free non-free-firmware" \
    --debian-installer none \
    --memtest none \
    --bootappend-live "boot=live quiet splash" \
    --iso-volume "NerfOS-$VERSION" \
    --image-name "nerf-os-$VERSION-$ARCH"

  echo "==> Running lb build (this takes 15-30 minutes)..."
  lb build

  ISO="nerf-os-$VERSION-$ARCH.iso"
  cp "$WORK_DIR/$ISO" "$OUTPUT/$ISO"
  echo "==> Done: $OUTPUT/$ISO"
  ls -lh "$OUTPUT/$ISO"
  ```

- [ ] **Step 1: Write 0010-antix-repo.hook.chroot**

  ```bash
  #!/bin/bash
  set -euo pipefail

  echo "==> Adding antiX 26 repository for runit packages"

  # antiX 26 is based on Debian Trixie
  # Repo URL verified from https://antixlinux.com/antix-26-released/
  ANTIX_REPO="deb http://repo.antixlinux.com/antix26 trixie main"
  ANTIX_KEY_URL="http://repo.antixlinux.com/antix26/antix26-archive-keyring.gpg"

  # Add keyring
  curl -fsSL "$ANTIX_KEY_URL" -o /usr/share/keyrings/antix26-archive-keyring.gpg

  # Add repo (pinned lower priority than Debian)
  echo "$ANTIX_REPO" > /etc/apt/sources.list.d/antix26.list
  cat > /etc/apt/preferences.d/antix26 << 'EOF'
  Package: *
  Pin: origin repo.antixlinux.com
  Pin-Priority: 100
  EOF

  apt-get update
  echo "==> antiX 26 repo added"
  ```

- [ ] **Step 1: Write 0020-runit-setup.hook.chroot**

  Create `build/config/common/hooks/live/0020-runit-setup.hook.chroot`:
  ```bash
  #!/bin/bash
  set -euo pipefail

  echo "==> Setting up runit as PID 1"

  # Install runit-init to replace systemd as PID 1
  apt-get install -y runit runit-init elogind libpam-elogind

  # Remove systemd if present (live-build minimal may not have it, but ensure clean)
  apt-get remove -y --purge systemd systemd-sysv libsystemd0 2>/dev/null || true
  apt-get autoremove -y 2>/dev/null || true

  # Ensure /sbin/init points to runit
  update-alternatives --set init /sbin/runit-init 2>/dev/null || true

  # Create runit stage directories if not present
  mkdir -p /etc/runit/1.d
  mkdir -p /etc/runit/3.d

  # Stage 1: system init script
  cat > /etc/runit/1 << 'EOF'
  #!/bin/bash
  # runit stage 1: system initialization
  export PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin

  # Mount essential filesystems
  mount -t proc proc /proc
  mount -t sysfs sysfs /sys
  mount -t devtmpfs devtmpfs /dev 2>/dev/null || true
  mount -t devpts devpts /dev/pts 2>/dev/null || true
  mount -t tmpfs tmpfs /run

  # Set hostname
  hostname -F /etc/hostname 2>/dev/null || true

  # Run stage 1.d scripts
  for script in /etc/runit/1.d/*; do
    [ -x "$script" ] && "$script"
  done
  EOF
  chmod +x /etc/runit/1

  # Stage 2: service supervision
  cat > /etc/runit/2 << 'EOF'
  #!/bin/bash
  # runit stage 2: start services
  export PATH=/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin

  # Detect boot mode from kernel cmdline
  /etc/runit/core-services

  # Start runsv for active services
  exec /sbin/runsvdir -P /etc/runit/runsvdir/default
  EOF
  chmod +x /etc/runit/2

  # Stage 3: shutdown
  cat > /etc/runit/3 << 'EOF'
  #!/bin/bash
  # runit stage 3: shutdown
  for script in /etc/runit/3.d/*; do
    [ -x "$script" ] && "$script"
  done
  EOF
  chmod +x /etc/runit/3

  echo "==> runit setup complete"
  ```

  ```bash
  #!/bin/bash
  set -euo pipefail

  echo "==> Configuring zram"

  apt-get install -y zram-tools

  # Configure zramswap: 50% of RAM, lz4 algorithm
  cat > /etc/default/zramswap << 'EOF'
  # NerfOS zram configuration
  # Algorithm: lz4 — fastest with good compression ratio
  ALGO=lz4
  # Size as percentage of total RAM
  PERCENT=50
  # Priority higher than disk swap
  PRIORITY=100
  EOF

  # Enable zramswap service (handled by zram-tools init script)
  # For runit, we use a direct activation in /etc/runit/1.d
  cat > /etc/runit/1.d/10-zram << 'EOF'
  #!/bin/bash
  # Start zram before any user processes
  if [ -f /etc/default/zramswap ]; then
    source /etc/default/zramswap
    TOTAL_RAM_KB=$(grep MemTotal /proc/meminfo | awk '{print $2}')
    ZRAM_SIZE_KB=$(( TOTAL_RAM_KB * PERCENT / 100 ))
    modprobe zram 2>/dev/null || true
    echo "$ALGO" > /sys/block/zram0/comp_algorithm 2>/dev/null || true
    echo "${ZRAM_SIZE_KB}K" > /sys/block/zram0/disksize 2>/dev/null || true
    mkswap /dev/zram0 2>/dev/null || true
    swapon -p "$PRIORITY" /dev/zram0 2>/dev/null || true
    echo "zram: activated ${ZRAM_SIZE_KB}K swap (${ALGO})"
  fi
  EOF
  chmod +x /etc/runit/1.d/10-zram

  echo "==> zram configured"
  ```

- [ ] **Step 3: Write nerf-optim**

  Create `scripts/nerf-optim`:
  ```bash
  #!/bin/bash
  # nerf-optim — NerfOS hardware detection and auto-optimization
  # Run at first boot or manually: sudo nerf-optim
  # Writes profile to /etc/nerf-os/hardware-profile

  set -euo pipefail

  PROFILE_DIR="/etc/nerf-os"
  PROFILE_FILE="$PROFILE_DIR/hardware-profile"
  FF_POLICY_DIR="/usr/share/nerf-os/firefox"

  # --- Pure functions (testable) ---

  detect_ram_tier() {
    local ram_kb="$1"
    if   [ "$ram_kb" -lt 786432  ]; then echo "low"   # < 768MB
    elif [ "$ram_kb" -lt 1572864 ]; then echo "mid"   # < 1.5GB
    else                                  echo "high"
    fi
  }

  calc_zram_size() {
    local ram_kb="$1"
    echo $(( ram_kb / 2 ))
  }

  get_browser_cache_mb() {
    local tier="$1"
    case "$tier" in
      low)  echo "64"  ;;
      mid)  echo "128" ;;
      high) echo "256" ;;
    esac
  }

  detect_storage_type() {
    local dev="${1:-sda}"
    local rotational
    rotational=$(cat "/sys/block/$dev/queue/rotational" 2>/dev/null || echo "1")
    if [ "$rotational" = "0" ]; then echo "ssd"; else echo "hdd"; fi
  }

  detect_gpu_vendor() {
    if lspci 2>/dev/null | grep -qi "nvidia"; then echo "nvidia"
    elif lspci 2>/dev/null | grep -qi "amd\|ati"; then echo "amd"
    elif lspci 2>/dev/null | grep -qi "intel"; then echo "intel"
    else echo "unknown"
    fi
  }

  detect_cpu_cores() {
    nproc 2>/dev/null || echo "1"
  }

  # --- Application logic ---

  apply_zram_config() {
    local ram_kb="$1"
    local zram_kb
    zram_kb=$(calc_zram_size "$ram_kb")

    cat > /etc/default/zramswap << EOF
  ALGO=lz4
  PERCENT=50
  PRIORITY=100
  EOF
    echo "zram: configured for ${zram_kb}KB"
  }

  apply_firefox_policy() {
    local cache_mb="$1"
    local hw_accel="$2"
    local webgl="$3"

    mkdir -p "$FF_POLICY_DIR"
    cat > "$FF_POLICY_DIR/policies.json" << EOF
  {
    "policies": {
      "DisableTelemetry": true,
      "DisableFirefoxStudies": true,
      "DisablePocket": true,
      "DisableCrashReporter": true,
      "HardwareAcceleration": $hw_accel,
      "Preferences": {
        "browser.cache.memory.capacity": { "Value": $((cache_mb * 1024)), "Status": "locked" },
        "webgl.disabled": { "Value": $webgl, "Status": "locked" },
        "browser.sessionstore.interval": { "Value": 60000, "Status": "locked" }
      }
    }
  }
  EOF
    echo "firefox: cache=${cache_mb}MB hw_accel=${hw_accel} webgl_disabled=${webgl}"
  }

  apply_io_scheduler() {
    local storage_type="$1"
    local scheduler
    if [ "$storage_type" = "ssd" ]; then
      scheduler="none"
    else
      scheduler="mq-deadline"
    fi
    # Write udev rule (takes effect on next boot or udevadm trigger)
    cat > /etc/udev/rules.d/60-nerfo-scheduler.rules << EOF
  ACTION=="add|change", KERNEL=="sd[a-z]*", ATTR{queue/rotational}=="1", ATTR{queue/scheduler}="mq-deadline"
  ACTION=="add|change", KERNEL=="sd[a-z]*", ATTR{queue/rotational}=="0", ATTR{queue/scheduler}="none"
  ACTION=="add|change", KERNEL=="nvme*", ATTR{queue/scheduler}="none"
  EOF
    echo "io_scheduler: $scheduler"
  }

  write_profile() {
    mkdir -p "$PROFILE_DIR"
    cat > "$PROFILE_FILE" << EOF
  # NerfOS hardware profile — generated by nerf-optim
  # Re-generate: sudo nerf-optim
  NERFO_RAM_TIER=$1
  NERFO_RAM_KB=$2
  NERFO_STORAGE=$3
  NERFO_GPU=$4
  NERFO_CPU_CORES=$5
  NERFO_DATE=$(date -u +%Y-%m-%dT%H:%M:%SZ)
  EOF
  }

  # --- Main execution (skipped in test mode) ---

  if [ -z "${NERF_OPTIM_TEST_MODE:-}" ]; then
    if [ "$(id -u)" -ne 0 ]; then
      echo "Error: nerf-optim must be run as root."
      exit 1
    fi

    echo "==> NerfOS hardware detection"

    RAM_KB=$(grep MemTotal /proc/meminfo | awk '{print $2}')
    RAM_TIER=$(detect_ram_tier "$RAM_KB")
    STORAGE=$(detect_storage_type "sda")
    GPU=$(detect_gpu_vendor)
    CORES=$(detect_cpu_cores)
    CACHE_MB=$(get_browser_cache_mb "$RAM_TIER")

    echo "  RAM: ${RAM_KB}KB ($RAM_TIER tier)"
    echo "  Storage: $STORAGE"
    echo "  GPU: $GPU"
    echo "  CPU cores: $CORES"

    # For old GPUs (pre-2012 Intel/ATI), disable hardware acceleration
    HW_ACCEL="true"
    WEBGL="false"
    if [ "$GPU" = "intel" ] || [ "$GPU" = "unknown" ]; then
      HW_ACCEL="false"
      WEBGL="true"
    fi

    apply_zram_config "$RAM_KB"
    apply_firefox_policy "$CACHE_MB" "$HW_ACCEL" "$WEBGL"
    apply_io_scheduler "$STORAGE"
    write_profile "$RAM_TIER" "$RAM_KB" "$STORAGE" "$GPU" "$CORES"

    echo "==> Profile written to $PROFILE_FILE"
    echo "==> Done. Reboot recommended."
  fi
  ```


- [ ] **Step 3: Write nerf-install**

  Create `scripts/nerf-install`:
  ```bash
  #!/bin/bash
  # nerf-install — NerfOS CLI installer
  # Copies the live system to a target disk and installs GRUB.

  set -euo pipefail

  VERSION="1.0"

  # --- Pure functions (testable) ---

  detect_firmware_mode() {
    local efivars_path="${1:-/sys/firmware/efi/efivars}"
    if [ -d "$efivars_path" ]; then echo "uefi"; else echo "bios"; fi
  }

  calc_swap_partition_gb() {
    local ram_mb="$1"
    if [ "$ram_mb" -lt 4096 ]; then echo "2"; else echo "0"; fi
  }

  validate_disk_input() {
    local disk="$1"
    if [ -z "$disk" ]; then
      echo "Error: no disk specified."
      return 1
    fi
    if [ ! -b "$disk" ] && [ -z "${NERF_INSTALL_TEST_MODE:-}" ]; then
      echo "Error: $disk is not a block device."
      return 1
    fi
    return 0
  }

  # --- UI helpers ---

  banner() {
    clear
    cat << 'EOF'

     _   _           __ ___  ____
    | \ | | ___ _ __|  _  / ___/ ___|
    |  \| |/ _ \ '__| |_) | |  \___ \
    | |\  |  __/ |  |  __/| |___ ___) |
    |_| \_|\___|_|  |_|    \____|____/

    NerfOS Installer v1.0
    ─────────────────────────────────

  EOF
  }

  confirm() {
    local msg="$1"
    local answer
    read -rp "$msg [yes/no]: " answer
    [ "$answer" = "yes" ]
  }

  # --- Installation steps ---

  step_select_disk() {
    echo ""
    echo "Available disks:"
    echo ""
    lsblk -d -o NAME,SIZE,MODEL --nodeps | grep -v "loop\|sr"
    echo ""
    read -rp "Enter target disk (e.g. /dev/sda): " TARGET_DISK
    validate_disk_input "$TARGET_DISK"

    echo ""
    echo "WARNING: ALL DATA on $TARGET_DISK will be PERMANENTLY ERASED."
    echo ""
    if ! confirm "Are you sure you want to install NerfOS on $TARGET_DISK?"; then
      echo "Installation cancelled."
      exit 0
    fi
  }

  step_partition_disk() {
    local disk="$1"
    local firmware="$2"
    local swap_gb="$3"

    echo ""
    echo "==> Partitioning $disk ($firmware mode, swap=${swap_gb}GB)..."

    wipefs -a "$disk"

    if [ "$firmware" = "uefi" ]; then
      # GPT: EFI (512MB) + swap (optional) + root (remaining)
      parted -s "$disk" mklabel gpt
      parted -s "$disk" mkpart EFI fat32 1MiB 513MiB
      parted -s "$disk" set 1 esp on
      if [ "$swap_gb" -gt 0 ]; then
        parted -s "$disk" mkpart swap linux-swap 513MiB "$((513 + swap_gb * 1024))MiB"
        parted -s "$disk" mkpart root ext4 "$((513 + swap_gb * 1024))MiB" 100%
        EFI_PART="${disk}1"
        SWAP_PART="${disk}2"
        ROOT_PART="${disk}3"
      else
        parted -s "$disk" mkpart root ext4 513MiB 100%
        EFI_PART="${disk}1"
        SWAP_PART=""
        ROOT_PART="${disk}2"
      fi
    else
      # MBR: swap (optional) + root (remaining)
      parted -s "$disk" mklabel msdos
      if [ "$swap_gb" -gt 0 ]; then
        parted -s "$disk" mkpart primary linux-swap 1MiB "$((1 + swap_gb * 1024))MiB"
        parted -s "$disk" mkpart primary ext4 "$((1 + swap_gb * 1024))MiB" 100%
        parted -s "$disk" set 2 boot on
        SWAP_PART="${disk}1"
        ROOT_PART="${disk}2"
        EFI_PART=""
      else
        parted -s "$disk" mkpart primary ext4 1MiB 100%
        parted -s "$disk" set 1 boot on
        ROOT_PART="${disk}1"
        SWAP_PART=""
        EFI_PART=""
      fi
    fi

    echo "    root: $ROOT_PART"
    [ -n "${SWAP_PART:-}" ] && echo "    swap: $SWAP_PART"
    [ -n "${EFI_PART:-}" ]  && echo "    efi:  $EFI_PART"
  }

  step_format() {
    echo "==> Formatting partitions..."
    mkfs.ext4 -F -L nerfo-root "$ROOT_PART"
    [ -n "${SWAP_PART:-}" ] && mkswap -L nerfo-swap "$SWAP_PART"
    [ -n "${EFI_PART:-}"  ] && mkfs.vfat -F32 "$EFI_PART"
  }

  step_copy_system() {
    echo "==> Copying live system to $ROOT_PART (this takes 5-15 minutes)..."
    mount "$ROOT_PART" /mnt
    [ -n "${EFI_PART:-}" ] && { mkdir -p /mnt/boot/efi; mount "$EFI_PART" /mnt/boot/efi; }

    rsync -aAX --progress \
      --exclude='/proc/*' \
      --exclude='/sys/*' \
      --exclude='/dev/*' \
      --exclude='/run/*' \
      --exclude='/tmp/*' \
      --exclude='/mnt/*' \
      --exclude='/media/*' \
      --exclude='/live' \
      / /mnt/

    mkdir -p /mnt/{proc,sys,dev,run,tmp,mnt,media}
  }

  step_generate_fstab() {
    echo "==> Generating /etc/fstab..."
    ROOT_UUID=$(blkid -s UUID -o value "$ROOT_PART")
    cat > /mnt/etc/fstab << EOF
  # NerfOS fstab
  UUID=$ROOT_UUID  /     ext4  defaults,noatime  0 1
  EOF
    if [ -n "${SWAP_PART:-}" ]; then
      SWAP_UUID=$(blkid -s UUID -o value "$SWAP_PART")
      echo "UUID=$SWAP_UUID  none  swap  sw  0 0" >> /mnt/etc/fstab
    fi
    if [ -n "${EFI_PART:-}" ]; then
      EFI_UUID=$(blkid -s UUID -o value "$EFI_PART")
      echo "UUID=$EFI_UUID  /boot/efi  vfat  umask=0077  0 1" >> /mnt/etc/fstab
    fi
  }

  step_setup_user() {
    echo ""
    read -rp "Username: " USERNAME
    while true; do
      read -rsp "Password: " PASSWORD; echo
      read -rsp "Confirm password: " PASSWORD2; echo
      [ "$PASSWORD" = "$PASSWORD2" ] && break
      echo "Passwords do not match. Try again."
    done
    read -rp "Hostname [nerfo]: " HOSTNAME
    HOSTNAME="${HOSTNAME:-nerfo}"

    echo "$HOSTNAME" > /mnt/etc/hostname
    cat > /mnt/etc/hosts << EOF
  127.0.0.1   localhost
  127.0.1.1   $HOSTNAME
  EOF

    # Create user in chroot
    chroot /mnt useradd -m -s /bin/bash -G audio,video,netdev,sudo "$USERNAME"
    echo "$USERNAME:$PASSWORD" | chroot /mnt chpasswd

    # Remove live user
    chroot /mnt userdel -r user 2>/dev/null || true
  }

  step_select_boot_mode() {
    echo ""
    echo "Default boot mode:"
    echo "  1) GUI (Openbox desktop)"
    echo "  2) Terminal (CLI only)"
    read -rp "Choice [1]: " BOOT_MODE_CHOICE
    BOOT_MODE_CHOICE="${BOOT_MODE_CHOICE:-1}"
    if [ "$BOOT_MODE_CHOICE" = "2" ]; then
      BOOT_MODE="terminal"
    else
      BOOT_MODE="gui"
    fi
  }

  step_install_grub() {
    local disk="$1"
    local firmware="$2"

    echo "==> Installing GRUB ($firmware)..."

    # Bind mounts for chroot
    mount --bind /dev  /mnt/dev
    mount --bind /proc /mnt/proc
    mount --bind /sys  /mnt/sys

    if [ "$firmware" = "uefi" ]; then
      chroot /mnt grub-install --target=x86_64-efi \
        --efi-directory=/boot/efi --bootloader-id=NerfOS
    else
      chroot /mnt grub-install --target=i386-pc "$disk"
    fi

    # Generate grub.cfg with correct default mode
    ROOT_UUID=$(blkid -s UUID -o value "$ROOT_PART")
    cat > /mnt/etc/default/grub << EOF
  GRUB_DEFAULT=0
  GRUB_TIMEOUT=5
  GRUB_DISTRIBUTOR="NerfOS"
  GRUB_CMDLINE_LINUX_DEFAULT="quiet splash nerf.mode=$BOOT_MODE"
  GRUB_CMDLINE_LINUX=""
  EOF

    chroot /mnt update-grub

    # Unmount bind mounts
    umount /mnt/dev /mnt/proc /mnt/sys
  }

  step_run_nerf_optim() {
    echo "==> Running hardware optimization..."
    chroot /mnt /usr/local/bin/nerf-optim || true
  }

  step_finish() {
    # Unmount
    [ -n "${EFI_PART:-}" ] && umount /mnt/boot/efi 2>/dev/null || true
    umount /mnt

    echo ""
    echo "╔══════════════════════════════════╗"
    echo "║  NerfOS installed successfully!  ║"
    echo "║                                  ║"
    echo "║  Remove the USB stick and reboot ║"
    echo "╚══════════════════════════════════╝"
    echo ""
  }

  # --- Main (skipped in test mode) ---

  if [ -z "${NERF_INSTALL_TEST_MODE:-}" ]; then
    if [ "$(id -u)" -ne 0 ]; then
      echo "Error: nerf-install must be run as root."
      exit 1
    fi

    banner

    RAM_MB=$(( $(grep MemTotal /proc/meminfo | awk '{print $2}') / 1024 ))
    FIRMWARE=$(detect_firmware_mode)
    SWAP_GB=$(calc_swap_partition_gb "$RAM_MB")

    echo "  Detected: ${RAM_MB}MB RAM, $FIRMWARE firmware"
    echo "  Partition plan: swap=${SWAP_GB}GB"

    step_select_disk
    step_select_boot_mode
    step_partition_disk "$TARGET_DISK" "$FIRMWARE" "$SWAP_GB"
    step_format
    step_copy_system
    step_generate_fstab
    step_setup_user
    step_install_grub "$TARGET_DISK" "$FIRMWARE"
    step_run_nerf_optim
    step_finish
  fi
  ```


## Troubleshooting Reference

**`lb build` fails at debootstrap:**
- Verify internet access from WSL2: `curl -I http://deb.debian.org`
- Verify running as root: `id` → should be `uid=0`

**antiX repo key download fails:**
- The key URL may change between antiX releases. Check https://antixlinux.com for the current signing key.
- Alternative: download key manually and place in `build/config/common/includes.chroot/usr/share/keyrings/`

**QEMU has no display (WSL2 headless):**
- Install an X server on Windows: VcXsrv or WSLg (Windows 11)
- Or use `-display none -nographic` for terminal-mode testing only

**Boot time exceeds 20 seconds in QEMU:**
- QEMU emulation is slower than real hardware. Measure on real hardware with a USB stick.
- Use `dd if=output/nerf-os-1.0-x86_64.iso of=/dev/sdX bs=4M status=progress` to write the USB.
